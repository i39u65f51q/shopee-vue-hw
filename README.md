## vue-hw

Vue3 蝦皮商品規格後台 - 前端作業

## 專案初始化

```sh
npm install
```

### 執行測試環境

```sh
npm run dev
```

### 編譯

```sh
npm run build
```

## 目錄結構

```
├── App.vue
├── Layout 頁面佈局
├── assets
├── classes 物件
├── components 共用元件
├── enum 列舉
├── main.ts
├── router 路由設定
├── stores 狀態儲存
├── types 型別定義
└── views 頁面
```
